El alumno tiene que trabajar sobre dos conceptos:
- Anteproyectos
- Proyectos

- Los anteproyectos tienen una estructura predefinida que se tiene que seguir.

- Los proyectos al ser el desarrollo de un software también se puede definir el ciclo de vida del mismo para definir el tipo de tareas a crear (desde el backend para la persistencia, pasando por los diferentes roles de usuario a las funcionalidades de la aplicación a los diferentes desarrollos de de la UI de usuario).

- El alumno puede o no crear tareas tanto en los anteproyectos cómo en los proyectos, en el caso de los anteproyectos las tareas podrían estar definidas bajo la estructura del modelo que tenemos de anteproyectos
