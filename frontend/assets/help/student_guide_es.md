# Guía rápida de uso (Estudiantes)

## Menú principal (barra superior y hamburguesa)
- Navega entre Anteproyectos, Proyectos, Tareas y Tablero Kanban.
- Usa el icono de idioma para cambiar español/inglés.
- El icono de campana muestra notificaciones.

## Flujo básico
1. Crea o revisa tu Anteproyecto desde Anteproyectos.
2. Cuando esté Aprobado, aparecerá como Proyecto.
3. En Proyectos verás pestañas: Detalles, Comentarios, Archivos, Kanban y Tareas.
4. Usa el Kanban para organizar tu trabajo y Tareas para hacer seguimiento.

## Consejos
- Mantén tus tareas actualizadas (estado y fechas).
- Revisa notificaciones con frecuencia.
- Ante dudas, consulta a tu tutor desde Comentarios del anteproyecto/proyecto.
